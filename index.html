Vintage

The starter kit for the true connoisseur.

Don't let Javascript fatigue get the best of you. We've sniffed through thousands of modules and their versions to bring you the best vintages.

Let's get started:

      npm install -g create-vintage-app

      create-vintage-app my-app
      cd my-app/
      npm start

What's included?
